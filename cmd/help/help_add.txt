用法：cbk add -n <任务名> -t <目标目录路径> [-b <备份存放路径>] [-k <保留数量>] [-bn <备份目录名>]

描述：
  添加一个新的备份任务。指定任务的基本信息，包括任务名、目标目录路径、备份存放路径、保留数量和备份目录名。

参数：
  -n <任务名>                必需。指定备份任务的名称。
  -t <目标目录路径>          必需。指定要备份的目标目录路径。
  -b <备份存放路径>          可选。指定备份文件存放的路径。默认路径为: "/用户家目录/.cbk/data/[目录名]"。
  -k <保留数量>              可选。指定备份文件的保留数量，默认值为3。
  -bn <备份目录名>           可选。指定备份目录的名称，默认为“目标目录名”。

示例：
  cbk add -n "任务1" -t "/home/user/documents"
  添加一个名为“任务1”的备份任务，目标目录为“/home/user/documents”，备份存放路径为默认路径，保留数量为默认值3，备份目录名为“documents”。

  cbk add -n "任务2" -t "/home/user/documents" -b "/home/user/backups" -k 5 -bn "backup_docs"
  添加一个名为“任务2”的备份任务，目标目录为“/home/user/documents”，备份存放路径为“/home/user/backups”，保留数量为5，备份目录名为“backup_docs”。

注意：
  1. 任务名和目标目录路径：任务名和目标目录路径是必需的，且任务名应具有唯一性。
  2. 备份存放路径：如果未指定备份存放路径，则使用默认路径。建议根据实际需求选择合适的备份存放路径。
  3. 保留数量：保留数量必须是一个正整数，建议根据实际需求合理设置。
  4. 备份目录名：如果未指定备份目录名，则默认使用目标目录的名称。
