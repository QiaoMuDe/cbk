用法：cbk edit -id <任务ID> [-n <任务名>] [-k <保留数量>]

描述：
  编辑指定备份任务的配置信息，包括任务名和保留的备份数量。

参数：
  -id <任务ID>       必需。指定要编辑的备份任务ID。
  -n <任务名>        可选。指定新的任务名。如果未指定，则任务名保持不变。
  -k <保留数量>      可选。指定备份文件的保留数量，默认值为3。如果未指定，则保留数量保持不变。
  -bn <备份目录名>   可选。指定新的备份目录名。如果未指定，则备份名保持不变。

示例：
  cbk edit -id 123 -n "新任务名"
  将任务ID为123的备份任务名修改为 "新任务名"，保留数量保持不变。

  cbk edit -id 123 -k 5
  将任务ID为123的备份任务的保留数量修改为5，任务名保持不变。

  cbk edit -id 123 -n "新任务名" -k 5
  将任务ID为123的备份任务名修改为 "新任务名"，并将保留数量修改为5。

  cbk edit -id 123 -bn "新备份目录名"
  将任务ID为123的备份任务的备份目录名修改为 "新备份目录名"，任务名和保留数量保持不变。

注意：
  1. 任务ID是必须的，否则无法确定要编辑的备份任务。
  2. 如果未指定任务名（-n）或保留数量（-k），则对应的配置项保持不变。
  3. 保留数量必须是一个正整数，建议根据实际需求合理设置。
